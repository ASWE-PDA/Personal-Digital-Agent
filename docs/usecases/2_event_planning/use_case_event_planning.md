# Use Case Description - Event Planning

| Systems | Luna, Smartphone |
| --- | --- |
| Use case | Event Planning |
| Actors | User, Luna |
| Description | When being called, Luna informs about new messages received and upcoming events for the day.The user can add new events into his calender and set reminders for existing events in his calender |
| Interfaces | Luna establishes connections to the API of the user's email provider, Outlook API   |
| Stimulus | User calls for events. / Luna sends push notification at a specific time. |
| Response | The data is sent from the repective APIs to the device and is being stored in memory. |
| Comments | - |
